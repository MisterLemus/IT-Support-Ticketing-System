## 🚀 Quick Start - Most Used Scripts

### Reset AD Password
```powershell
.\scripts\ad-user-management.ps1 -Action ResetPassword -Username jsmith
```

### Run System Diagnostics
```powershell
.\scripts\system-diagnostics.ps1 -ComputerName WORKSTATION-042
```

### Network Troubleshooting
```powershell
.\scripts\network-troubleshoot.ps1 -Target 192.168.1.1
```
